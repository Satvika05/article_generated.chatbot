# 🧠 Article Generator Chatbot using Open-Source LLMs

This project uses three open-source LLMs to generate articles based on user prompts:

- `gptJ`
- `falcon`
- `mistral`

## 🚀 Features

- Generates articles using Transformers
- Streamlit-based UI
- Easy comparison of model outputs

## 📦 How to Run

### 📍 google colab

Open `Article_Generator_LLMs.ipynb` and run each cell step-by-step to generate outputs.

### 🌐 Streamlit App

Run the Streamlit app using:

```bash
streamlit run streamlit_app.py
